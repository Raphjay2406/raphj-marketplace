# Genorah v4.2.0 — "Visual Selection"

## Visual Selection Companion
- Activates the existing companion server for the spec/design/iterate steps.
- Deterministic renderer (`scripts/companion/render-screen.mjs`) turns a screen-spec into color-accurate, clickable HTML cards: real DNA hex swatches, real fonts, colored HTML mockups (no more monochrome ASCII), per-option hero.
- Four surfaces: creative directions, iterate proposals, archetype pick, palette/DNA (with WCAG contrast badges).
- Per-option hero images via gpt-image on directions/iterate, with a DNA-gradient fallback when unavailable.
- Selection round-trip: clicks are recorded to `.events`; `read-selection.mjs` resolves the pick for the next turn. ASCII remains the no-companion fallback.
- `start-server.sh --open` auto-opens the browser (best-effort).
