# Industry Rules Data Catalog

> 52 industry-specific reasoning rules for landing page design.
> Each rule maps to Genorah archetypes, color moods, and anti-patterns.

---

## Industry: SaaS (General)
- **Category:** Tech
- **Recommended Pattern:** Hero with product screenshot + CTA -> Social proof logos -> 3-feature grid -> Interactive demo/video -> Pricing table -> Testimonials -> FAQ -> Final CTA
- **Style Priority:** Clean, professional, trust-first, conversion-optimized
- **Color Mood:** blue trust, indigo focus, neutral gray
- **Typography Mood:** geometric sans, clean weights, high readability
- **Key Effects:** Subtle scroll reveals, product UI hover states, gradient accents
- **Anti-Patterns:** Cluttered hero, more than 2 CTAs above fold, stock photos of smiling people, autoplay video
- **Archetype Affinity:** Neo-Corporate, Swiss/International, AI-Native
- **Severity:** HIGH

## Industry: Micro SaaS
- **Category:** Tech
- **Recommended Pattern:** Bold headline + inline demo -> Problem statement -> Before/after -> 1-2 key features -> Founder social proof -> Simple pricing (1-2 tiers) -> CTA
- **Style Priority:** Personal, direct, founder-led, opinionated
- **Color Mood:** vibrant accent on neutral, single highlight color
- **Typography Mood:** friendly sans, slightly rounded, conversational
- **Key Effects:** Minimal animations, fast load, inline demos
- **Anti-Patterns:** Enterprise-style feature grids, complex pricing, corporate language, fake urgency
- **Archetype Affinity:** Playful/Startup, Neubrutalism
- **Severity:** MEDIUM
