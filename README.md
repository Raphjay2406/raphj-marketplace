# raphj-marketplace

Personal Claude Code plugin marketplace by Raphjay2406.

## Installation

```bash
claude plugin marketplace add https://github.com/Raphjay2406/raphj-marketplace
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `v0-ahh-skill` | v0-inspired frontend design skill for React/Next.js with shadcn/ui and Tailwind CSS |

## Install a Plugin

After adding the marketplace, install any plugin:

```bash
claude plugin install v0-ahh-skill@raphj-marketplace
```
