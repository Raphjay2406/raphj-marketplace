# /gen:signature-mark

Forges a bespoke 3D mark from brand essence; registers in the uniqueness ledger.

## Workflow

1. Read DESIGN-DNA.md `brand_essence` + `project_id`.
2. Call SignatureForge (Rodin provider).
3. Write public/assets/signature-mark.glb + register in `~/.claude/genorah/signatures.db`.
4. Print collision-check status.
