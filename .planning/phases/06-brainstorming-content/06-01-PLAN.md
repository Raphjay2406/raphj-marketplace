**Trigger and Role** (~5 lines):
Trigger: during brainstorm phase, when generating creative directions, when exploring design concepts, when studying competitive landscape, design exploration, visual direction.
Role: "You are a creative director who never designs from a blank slate. Every creative decision is grounded in competitive research, cross-industry inspiration, and strategic differentiation. You study the landscape, identify opportunities, and generate directions that are both creatively bold and strategically sound."

**Layer 1: Decision Guidance — The Brainstorm Protocol** (~80-100 lines):
Define the complete 7-phase brainstorm process. This is the CORE of the skill — a prescriptive protocol, not advisory prose.

**Phase 1: Brief Extraction** (~10 lines)
Before any research, extract from project context:
- Industry vertical (maps to curated library below)
- Brand personality (4-6 adjectives)
- Target audience (who must be impressed)
- Emotional response target (what visitors should FEEL)
- Known constraints (existing brand, technical requirements)
- Competitive landscape (who are we differentiating from)

**Phase 2: Industry Research** (~15 lines)
Protocol:
1. Look up the project's industry in the Curated Industry Library below
2. Study 6-8 reference sites from the curated library for the relevant vertical
3. If web search is available: supplement with 2-3 CURRENT award winners from Awwwards/SOTD in the same industry
4. Include 2-3 references from OUTSIDE the industry (pull from cross-pollination skill's pairing suggestions for this vertical)
5. For each reference site, run the Competitive Teardown (Phase 3)
6. Output: structured research findings in `.planning/modulo/research/DESIGN-REFERENCES.md`

**Phase 3: Competitive Teardown** (~20 lines)
For EACH reference site analyzed, produce a structured teardown. The teardown template parallels the existing start-design reference analysis but adds content strategy and UX flow dimensions:

```markdown
### [Site Name] — [URL]
**Industry:** [vertical] | **Relevance:** [direct competitor / adjacent / outside-industry inspiration]

**Visual Patterns:**
- Color approach: [palette character, dark/light, accent strategy]
- Typography: [font choices, hierarchy approach, display font character]
- Layout: [grid structure, breaking points, asymmetry vs. symmetry]
- Depth: [shadows, layers, glass effects, texture]
- Motion: [intensity level (1-5), signature motion, scroll behavior]
- Signature element: [what makes this site visually memorable]

**Content Strategy:**
- Headline style: [declarative/question/provocative/minimal]
- CTA philosophy: [aggressive/subtle/absent/progressive]
- Social proof: [placement, specificity, format]
- Narrative approach: [story-driven/feature-list/demo-first/manifesto]
- Voice personality: [formal/casual/technical/warm]

**UX Flow Analysis:**
- First impression: [what captures attention in first 2 seconds]
- Scroll depth: [estimated sections, pace, content density]
- Conversion path: [steps to primary action]
- Mobile strategy: [simplified/adapted/different experience]

**Key Takeaways for Our Project:**
1. [What to learn from this site]
2. [What to avoid/differentiate from]
3. [Opportunity this site doesn't exploit]
```

**Phase 4: Archetype Shortlisting** (~10 lines)
Based on research findings:
1. Identify which 4-5 archetypes from design-archetypes skill align with the project's personality and audience
2. Cross-reference with what competitors are NOT doing (differentiation opportunity)
3. Shortlist 3 archetypes for direction generation — one should be an unexpected/"bold" choice
4. Reference the cross-pollination skill for each archetype's pairing suggestions

**Phase 5: Direction Generation** (~10 lines)
For each of the 3 shortlisted archetypes:
1. Build a full concept board following the creative-direction-format skill's template
2. Include cross-pollination elements from the cross-pollination skill (1-2 constraint breaks per direction, based on boldness calibration)
3. Create ASCII mockups following the creative-direction-format skill's ASCII specification
4. Run the distinctness validation matrix from the creative-direction-format skill
5. If validation fails: regenerate the most similar direction

**Phase 6: User Selection and Mixing** (~10 lines)
1. Present 3 directions using the creative-direction-format skill's presentation structure
2. User selects one direction OR cherry-picks elements across directions
3. If mixing: follow the creative-direction-format skill's free mixing protocol
4. Lock the final direction into Design DNA format (reference design-dna skill)

**Phase 7: Brand Voice Generation** (~10 lines)
After direction is selected:
1. Trigger the copy-intelligence skill's brand voice document template
2. Generate BRAND-VOICE.md following the template, populated with the selected archetype's voice personality
3. Save to `.planning/modulo/BRAND-VOICE.md`
4. This runs as the FINAL step of brainstorm, before section planning begins

**Layer 2: Curated Industry Reference Library** (~250-300 lines):
Cover 12 industry verticals. For EACH vertical, provide:

1. **Vertical name** (heading)
2. **6-8 exemplary sites** with pattern-based descriptions. Format:
   "[Site Name] — [pattern description: what specific design pattern they demonstrate and WHY it's award-winning, described so the pattern survives site redesigns. 2-3 sentences.]"
3. **Visual landscape summary** (2-3 sentences: what the industry looks like collectively, common patterns, dominant aesthetics)
4. **Content patterns** (1-2 sentences: how this industry typically handles copy, CTAs, social proof)
5. **Differentiation opportunities** (2-3 bullets: what most sites in this industry DON'T do, which is where creative directions can stand out)

The 12 verticals:
1. SaaS / Developer Tools
2. E-commerce / Fashion
3. E-commerce / General Retail
4. Creative Agency / Design Studio
5. Portfolio / Personal
6. Blog / Publication / Media
7. Health / Wellness
8. Food / Beverage / Restaurant
9. Fintech / Dashboard
10. Architecture / Interior Design
11. Education / Social Impact
12. Gaming / Entertainment

CRITICAL QUALITY RULES:
- Reference sites MUST be real, award-winning sites that a creative director would actually study (Awwwards SOTD winners, FWA winners, Dribbble featured)
- Pattern descriptions MUST focus on the observable PATTERN, not just the site name. "Linear demonstrates developer-tool elegance: dark mode with restrained accent colors, monospace micro-detail in labels, and product-as-hero with a live interactive demo above the fold" — NOT just "Linear is a good SaaS site"
- Every vertical MUST have at least 1 reference from outside the typical industry aesthetic (the "unexpected" reference)
- Differentiation opportunities should be specific and actionable, not vague ("most SaaS sites ignore editorial typography" is better than "be creative")

**Multi-Industry Projects** (~10 lines):
When a project spans multiple industries (e.g., "a SaaS product for restaurants"):
1. Identify primary and secondary verticals
2. Research both verticals (6-8 sites from primary, 3-4 from secondary)
3. Cross-pollination comes from OUTSIDE both verticals (not between them — that's adjacent, not distant)
4. Archetype shortlisting considers both audience expectations
5. Direction generation emphasizes the primary vertical's audience while borrowing depth from the secondary

**Layer 4: Anti-Patterns** (~50-60 lines):
6 anti-patterns (this skill is complex enough to warrant 6):

1. **Blank Slate Ideation**: Starting creative direction generation without studying any reference sites. Instead: ALWAYS complete Phases 1-3 (Brief, Research, Teardowns) before Phase 4 (Shortlisting). Every direction must cite at least 2 specific references that influenced it.

2. **Reference Tourism**: Listing 10 sites without actually analyzing them. "Linear is cool" is not a teardown. Instead: every reference gets the full 3-dimension teardown (Visual Patterns + Content Strategy + UX Flow) with specific "Key Takeaways for Our Project."

3. **Safe Shortlisting**: Choosing the 3 most obvious archetypes for the industry (SaaS = Neo-Corporate, Swiss, Data-Dense). Instead: at least one shortlisted archetype should be an unexpected choice that research revealed could work. The unexpected choice is often the most award-winning direction.

4. **Copy-Paste Archetype**: Taking an archetype's locked palette and techniques verbatim without any project-specific adaptation or cross-pollination. Instead: every direction must include at least 1 cross-pollination element that makes it unique to THIS project, not generic archetype output.

5. **Thin Directions**: Presenting directions with just color palette + font + vague description (the old v6.1.0 pattern). Instead: every direction uses the full creative-direction-format concept board template with all 11 sections including ASCII mockups, motion identity, and tension plan.

6. **Forgetting Phase 7**: Generating great creative directions but skipping brand voice generation. Instead: Phase 7 (brand voice via copy-intelligence) is MANDATORY. Brainstorm is incomplete until BRAND-VOICE.md exists alongside BRAINSTORM.md.

Ensure the COMPLETE skill is 600-800 lines total across both tasks. The skill should read as a single cohesive document.
