**Embedded Beat Parameter Table (HARD CONSTRAINTS):**
Copy the full beat parameter table from v6.1.0's section-builder.md (lines 42-53). These are the same values. Include all 10 beat types with Height, Density, Anim Intensity, Whitespace, Type Scale, Layout columns.

**Embedded Quality Rules (do NOT read skill files):**

Anti-Slop Quick Check (5 items -- builder self-checks before finishing):
1. ONLY DNA color tokens used? (No raw hex, no Tailwind color defaults like blue-500)
2. DNA fonts used? (No system defaults like Inter, Arial, sans-serif)
3. DNA spacing scale used? (No arbitrary values like gap-3, p-7)
4. Beat parameters met? (Check height, density, whitespace against table)
5. Signature element present? (If assigned in spawn prompt)

Performance Rules (embedded):
- Images: next/image with width/height, lazy loading below fold, WebP/AVIF format
- Animations: CSS for simple (opacity, transform), JS only for complex. prefers-reduced-motion on ALL animations
- Fonts: next/font for loading. Display: swap
- No inline styles. Tailwind classes only (using DNA tokens via CSS custom properties)

Micro-Copy Rules (embedded):
- BANNED phrases: "Submit", "Learn More", "Click Here", "Get Started" (unless spawn prompt content explicitly uses them)
- Every CTA must be specific to the action: "See Pricing", "Book a Demo", "Read the Case Study"
- Placeholder text is NEVER acceptable. All copy comes from spawn prompt content section

**Build Process:**
1. Read spawn prompt context (already in prompt -- do not re-read files)
2. Read your PLAN.md at the specified path
3. Execute tasks sequentially from PLAN.md
4. For each task: implement, then verify against must_haves
5. After all tasks: run light auto-polish pass
6. Self-verify: beat parameters, anti-slop quick check, DNA compliance
7. Write SUMMARY.md

**Light Auto-Polish Pass (mandatory final stage):**
Before writing SUMMARY.md, verify and add if missing:
- All interactive elements have hover/focus/active states
- Subtle transforms on interactive elements (scale 1.02-1.05 on hover)
- Texture application per DNA spec (if archetype uses textures)
- Smooth scroll integration for anchor links
- prefers-reduced-motion variant for ALL animations
- Custom selection color matching DNA accent
- Focus-visible outlines using DNA accent color
