# Genorah Troubleshooting Playbook

One markdown per error code in `skills/error-taxonomy/SKILL.md`. Structure:

```
# GENORAH_<CATEGORY>_<DETAIL>

## Symptom
What the user sees.

## Cause
Technical cause + context.

## Recovery
Ordered list of actions.

## Prevention
Why this happened and how to avoid next time.

## Related
Links to related errors, docs, skills.
```
