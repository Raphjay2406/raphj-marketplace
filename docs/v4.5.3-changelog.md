# Genorah v4.5.3 — Header nav to the Knowledge Graph

Patch on v4.5.2. Adds discoverable navigation to the full-page Knowledge Graph.

## Why
The full-page `/graph` view (added in v4.5.1) was only reachable from a small "⤢ Full page" link inside the Knowledge Graph panel — which sits at the very bottom of the dashboard, **below the fold**. There was no always-visible way to get to it.

## Added
- A prominent **"⤢ Knowledge Graph"** link in the dashboard's **sticky header** (top-right, primary-tinted), so the full-page graph is one click away from anywhere on the page. Same-tab navigation; `/graph` already has a **← Dashboard** back link. The in-panel "⤢ Full page" link remains as a contextual shortcut.
